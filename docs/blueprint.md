# **App Name**: TeamsFlow

## Core Features:

- Kanban Board Integration: Adds a button to the Teams web interface that opens a Kanban board within the extension.
- Conversation Card Creation: Allows users to convert Teams conversations into Kanban cards.
- Customizable Columns: Enables users to create Kanban columns with custom names, colors, and icons. The storage is on the client side, only.
- Quick Response Trigger: Adds a quick response trigger (/) in Teams conversations.
- Quick Response Management: Allows users to manage and customize quick responses within the extension settings.
- AI-Powered Task Prioritization: The extension includes an optional AI powered "tool" that intelligently suggests task priorities based on conversation content and frequency.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to align with Teams' aesthetic but with a distinct identity.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Accent color: Electric green (#7CFC00) for interactive elements and highlights to draw attention.
- Body and headline font: 'Inter', a sans-serif font known for its clean, readable design.
- Use flat, minimalistic icons for Kanban cards and column customization.
- Kanban board layout with drag-and-drop functionality for moving conversation cards between columns.
- Subtle animations for card transitions and column creation.
