# TeamsFlow - Extensão para Microsoft Teams

Este projeto é uma extensão do Chrome que funciona como um motor de produtividade para o Microsoft Teams Web.

## Como testar a Extensão no seu Chrome:

1.  Abra o Google Chrome.
2.  Vá para `chrome://extensions/`.
3.  No canto superior direito, ative o **Modo do desenvolvedor**.
4.  Clique no botão **Carregar sem compactação** (Load unpacked).
5.  Selecione a **pasta raiz deste projeto** (a pasta que contém este arquivo `README.md`).
6.  Agora, abra o [Microsoft Teams Web](https://teams.microsoft.com/) no seu navegador.
7.  Teste as funcionalidades, como o quadro Kanban e as respostas rápidas.

## Funcionalidades:
- **Quadro Kanban**: Permite organizar tarefas.
- **Respostas Rápidas**: Crie atalhos para inserir textos longos rapidamente.

Todo o armazenamento de dados é feito localmente usando o `chrome.storage.local`.
